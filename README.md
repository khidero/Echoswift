# Echo Swift - Denver Junk Removal

A modern, professional website for Echo Swift, a Denver-based junk removal service.

## Features

- Modern, responsive design with emerald/teal color scheme
- Smooth animations and transitions
- Service accordion with detailed descriptions
- Contact form for quote requests
- Mobile-friendly navigation
- Eco-friendly messaging (70% recycled/donated)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
```

## Tech Stack

- React 19
- TypeScript
- Vite
- TailwindCSS
- Lucide React Icons
- React Router DOM

## Color Scheme

- Primary: Emerald (from-emerald-500 to-teal-600)
